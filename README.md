# WAHSUN Blog

WAHSUN 的個人部落格，記錄技術與生活。

使用 [Astro](https://astro.build/) 與 [Tailwind CSS](https://tailwindcss.com/) 建置，基於 [Microblog](https://github.com/michael-andreuzza/microblog) 主題修改。

## 開發

```bash
npm install
npm run dev      # 啟動開發伺服器，預設 http://localhost:4321
npm run build    # 建置靜態檔案到 dist/
npm run preview  # 預覽建置結果
```

## 部署

靜態網站部署至 GitHub Pages，`blog` 分支為建置輸出。
